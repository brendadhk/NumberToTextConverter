#NumberToTextConverter
=====================
C# Windows App to simulate the way a cheque program converts a decimal number into the corresponding text. 
For example:
Input:          “123.45”
Output:        “ONE HUNDRED AND TWENTY-THREE DOLLARS AND FORTY-FIVE CENTS”
